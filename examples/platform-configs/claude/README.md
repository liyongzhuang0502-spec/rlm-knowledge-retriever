# Claude Code MCP Integration

## Setup

1. Copy `mcp-config.json` to your Claude Code config:

```bash
cp mcp-config.json ~/.claude/claude_desktop_config.json
```

2. Set your OpenAI API key:

```bash
export OPENAI_API_KEY=your-key
```

3. Restart Claude Code

## Usage

In Claude Code, the MCP tools will be available:

```
Use the rlm-knowledge-retriever MCP server to search the knowledge base for JWT authentication patterns.
```
